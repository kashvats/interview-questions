2. CODE: SECURE PASSWORD HASHING (Python) -- ```python
import bcrypt

def hash_password(password: str):
 # Salt adds randomness to the hash
 salt = bcrypt.gensalt()
 return bcrypt.hashpw(password.encode('utf-8'), salt)

def check_password(plain_password: str, hashed_password: bytes):
 return bcrypt.checkpw(plain_password.encode('utf-8'), hashed_password)

# Staff level tip: Use Argon2 for better protection against specialized hardware (ASICs).
```
