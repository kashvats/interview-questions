- **Implicit Dependency**: Jab ek resource doosre resource ka attribute use karta hai (e.g., EC2 instance VPC ki ID use kare). Terraform ko pata hai ki pehle VPC banani hai.

2. CODE: DYNAMIC SECURITY GROUP RULES -- ```hcl
variable "ingress_rules" {
 type = list(number)
 default = [80, 443, 8080]
}

resource "aws_security_group" "zomato_sg" {
 name = "zomato-web-sg"

 dynamic "ingress" {
 for_each = var.ingress_rules
 content {
 from_port = ingress.value
 to_port = ingress.value
 protocol = "tcp"
 cidr_blocks = ["0.0.0.0/0"]
 }
 }
}
```
